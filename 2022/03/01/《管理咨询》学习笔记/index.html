《管理咨询》学习笔记第一章——咨询到底是什么？

互联时代，知识浩如烟海，很高兴本笔记能被您阅读到。这个博客专题记录了本人在天津大学研究生在读期间选修《管理咨询》课程时的学习笔记，之中加入了部分个人见解和阅读资料，仅供个人参考使用。

原课程来自中国mooc，是一门由暨南大学管理学院院长李从东讲授的国家一流学科建设精品课程，李从东老师也曾在天津大学经管学部从教。

1-1 案例：一个世界级的企业是如何形成的

  • 管理咨询是什么？

    答：管理咨询师利用自己掌握的知识，为企业的需求提供的管理智力服务，管理支持服务
    注：管理咨询师包括管理顾问公司的管理咨询师和企业的管理人员，面向问题包括如何诊断一个企业、如何去纠正企业的错误、如何去弥补企业的漏洞、如何去维护企业的健康

  • 华为如何解决职能部门孤岛效应

    • 1998年到2003年期间引进IBM进行8项项目管理业务：IPD（集成产品开发）、ISC（集成供应链）等
    • 绩效文化
    • 正确的战略方向

1-2 管理咨询的理论依据——负熵流与系统稳定

企业的成长进程
无论大小企业，都面临着错综复杂的企业管理问题，以至于破产，大型国际企业以雷曼兄弟为例，中小企业则以肖克利电子公司等为例；但也不乏克服了管理问题，最终走出困境的企业案例。

透过现象看本质，高等教育强调学生对理论知识的理解与应用，那么企业管理中有哪些主要理论呢？
其中之一便是 “熵增”现象
普利高金耗散结构理论：
封闭系统总是无序的，熵增是系统混乱不断增长的现象
稳定是无需的极端形态（热寂）

熵增的分类：

内生性熵增：KL推行质量责任追究制让员工缺乏责任感，进而引发推诿扯皮、MD作业长调整生产节拍、主观随意的调整破坏了企业生产流程。

外源性熵增：行业断崖式下滑、恶化的营商环境。

企业发展的规律——走向消亡，或在持续的再平衡中发展
封闭系统最终回导致熵爆（热寂）——有效能量全部转化为无效能量后，系统达到平衡
解决路径：导入负熵流

热力学第一定律：能量守恒
热力学第二定律：能量的总和+有效能量+无效能量（系统的熵增是一个常态）
负熵流
负熵：系统有序化、组织化、复杂化状态的一种度量
管理咨询属于软投入，其功能即有目的地导入负熵流
负熵流导致系统涨落——>系统远离平衡态——>系统创新或组织发展

1-3 企业需要导入负熵流的几种情况

具体情况具体治理

  • 有病而不自知，知病而不能自治

  • 创新乏力——组织遭遇业绩天花板

  • 屡遭败绩——组织需要外力介入遏制下滑趋势

  • 外部环境突然变化——组织处在迷茫状态或战略分歧点

  • 某方面遇到瓶颈，自身无力突破

  • 缺乏系统化的管理设计思想和管理改善方案

管理咨询的负熵流作用

  • 上世纪八十年代韦尔奇领导通用电气改革
  • 上世纪九十年代郭士纳出任IBM总裁，推进IBM战略转型

管理咨询三大好处

  • 专业能量和专家力量
    • 宽阔的行业事业和众多企业变革经验
    • 树术业有专攻
    • 咨询公司的跨行业经验有利于客户管理创新
  • 第三方中立左右和客观的力量
    • 企业管理者本身属于局中人，自己给自己设计改革方案，很难客观、公正，往往看不到问题症结，或难以服众，此时必须有第三方介入。
  • 变革所需的外部推动力量
    • 客服自身的弱点或惰性

1-4 警惕负熵流的不当导入

  • 经验主义

  • 教条主义

  • 机会主义

  • 合法与非法

    • 安达信陷入安然财务丑闻
    • 毕马威在新华工程财报中作假
    • 德勤陷入科龙假账丑闻

  • 基础课与专业课

    • 麦肯锡兵败实达
      原因分析：方案过于激进，世达公司基础课尚未几个；未作十点，匆忙推出
    • 康佳、乐百氏

  • 传教士与张天师

  • 规范的管理咨询：培训、合作——知识转移，道、理、法、术均要传递到位

  • 但很多企业希望咨询公司包办一切，只要产出指标达到要求即可

  • 标准化与个性化

    • 罗兰贝格的《南航心约》与山寨版的“XX心约”

1-5 管理咨询业的前世今生——咨询行业的历史与现状

1. 管理咨询：古老的行业

  • 春秋战国：商鞅、孙武、管仲、孙膑、范蠡

  • 秦汉三国：范增、张良、陈平、诸葛亮、鲁肃、郭嘉

  • 清代：咨询行业发展成一个行业组织，如曾国藩的湘军幕府、左宗棠的楚军幕府、李鸿章的淮军幕府。

与现代管理咨询业的本质区别：中国古代的管理咨询师是为政治信仰而非商业价值工作。

2. 现代管理咨询业

  • 1926年，詹姆斯·麦肯锡成立了一个会计与管理工程事务所，麦肯锡公司成立。

  • 1939年，Marvin Bower买下麦肯锡公司，将其改造为现代管理咨询公司。

  • 1973年，威廉姆·贝恩离开了BCG（波士顿咨询公司），成立了贝恩公司。

由此，形成了麦肯锡、波士顿、贝恩三大咨询公司，分为两大派：

  • 麦肯锡：做咨询一定不要过多的插手客户内部的事务

  • 贝恩：咨询与实施相统一，即“关系咨询法”

3. 世界咨询业的主要代表

IBM全球企业服务部、埃森哲、毕博、凯捷、德勤（世界上最新的五大咨询商）、麦肯锡、波士顿、罗兰·贝格、贝恩、博思等

4. 本土管理咨询业

汉普、远卓、北大纵横、高盛咨询等

5. 访谈：高盛咨询-丁佰胜

1）本土咨询业在竞争中的特点？

市场的巨大需求为外资和本土咨询业提供了同样的机会。

2）相比外资大品牌，本土咨询业的优势？

外资与本土咨询公司客户群体不同。外资品牌的客户群体基本上是以中国大型国营企业、外资企业和超大型的民营企业为主，本土的咨询公司大部分是以中小型民营企业为主。但随着社会的发展，这种情况发生了变化，双方的客户群体会相互渗透。

二者主要区别在于咨询模式的需求，大型外资企业的咨询公司更多的是重在方案，本土的咨询公司注重咨询和实施相统一。

6. 具有管理咨询能力的运营与供应链管理领域的职业管理者

首席运营官、工厂经理、项目经理、效率分析师

1-6 管理咨询的属性和规则——分类、任务、过程、要求

1. 分类

1）广义和狭义

  • 广义的管理咨询（商业咨询）：战略与商业模式咨询、资产与财务咨询、市场策略咨询

  • 狭义的管理咨询：业务流程与组织管理咨询、人力效能与人力资源管理咨询、运营管理咨询、信息化与集成化管理咨询

2）按咨询方法分类

  • 访谈、工作日写实、调查问卷等方式，咨询师根据经验进行分析，类似中医的“望、闻、问、切”

  • 大量搜集各种数据、报表，咨询师根据模型、仿真进行分析，类似于西医的“放射科”“B超室”“化验室”

2. 管理咨询的内涵、任务和使命

1）内涵：管理咨询是具有丰富的管理知识和经验并且掌握了咨询技法的人所从事的高智能的服务事业。

2）任务和使命：

  • 咨询人员在委托者提出要求的基础上（需求导向或服务导向），
  • 深入对象组织与对象组织的管理人员密切结合（没有合作就没有成功），
  • 应用科学的方法，找出对象组织存在的主要问题（找到真正的问题是改善最重要的前提），
  • 进行定量和确有论据的定性分析，找出存在问题的原因（去伪存真），
  • 提出切实可行的改善方案（规划与实施的一致性、分离性），
  • 进而培训对象组织的人员，指导实施方案（谁来执行？创伤的愈合靠的是病人自身），
  • 使对象组织的资源配置和要素耦合更加合理，运行体制、运行机制、运行制度得到改善（尽量解决高层级问题），
  • 从而提高对象组织的管理水平和运作效益（最终输出应能量化和显示化）。

3. 管理咨询的原则

①有限性：目标有限、范围有限、参与程度有限（有限→有重点→有突破）

②客观性：管理咨询师应聚焦于问题和方案，确保方案合理、缜密；对象组织自行决定时机、步骤、力度、角度

③选择性：在“问题-分析-方案”的路径中，存在多目标效价、多种路径

④妥协性：在必要性与可能性、长远目标与阶段目标、理想模型与约束条件之间寻找平衡点。有效管控咨询主客体之间、决策层与执行层之间、不同部门之间的矛盾，整合分歧

⑤辅助性：对象组织既是管理咨询的委托者，也是咨询结果的采纳者、实施者和风险承担者。咨询师应尽力协助对象组织，而不是越俎代庖。

4. 管理咨询的主客体关系

1）主体：咨询者，其职业被称为管理咨询师

素质要求：①深厚和宽广的知识基础

​ ②具有丰富的管理实践经验

​ ③具有经验知识化和知识集成化的能力

​ ④具有好学、务实、严谨、细致的工作作风

​ ⑤重视来自一线的事实、数据和各个层次的声音

​ ⑥尊重委托者、尊重同行、尊重同事

​ ⑦遵守与委托方达成的保密协议，诚实守信，善于与各个

方面合作

​ ⑧具有系统思维，能够自顶向下的进行规划和设计，能够

自底向上的由点到面的实施方案

​ ⑨尊重科学，实事求是，敢于面对不尊重事实的任何压

​ 力，“不屈吾道”。重视科学方法、技术工具的使用

责任：①协助委托人（企业）认识其发展中的各种矛盾及其运动规律

（培训、辅导）

​ ②在委托人（企业）主要矛盾周期的不同阶段，协助企业分别采

取系统预防、系统预警、系统辨识、系统处置的方式，最大

限度地保障企业的利益，使企业得以持续、健康发展

③以委托人的长远利益为目标，逐步培育其自我诊断、自我修

复的能力

2）客体：对象组织

如果对象组织是企业，管理咨询指的是企业管理咨询

要求：

①组织必须有意愿寻求并接受管理咨询，深信管理咨询可以促进组织变革、推进组织发展。

②组织是变革的主体，必须设定变革的愿景，主动以高层主导、团队参与的方式接受管理咨询。

③组织不得讳疾忌医，以虚假事实和资料误导咨询者

④组织不得违反咨询程序，将主观意志强加给咨询者

⑤组织不能“得病乱求医”，或良莠不分，或断章取义，或盲从咨询建议

【思考题】以实例说明管理咨询原则的合理性

1-7 管理咨询的流程和步骤——诊与疗的关系

1. 企业诊疗的步骤

第一步：数据分析与望闻问切，确认病本

第二步：总体规划，制订咨询计划

第三步：分析主要问题并研究解决方案

第四步：培训队伍，授人以渔

第五步：试点，修正方案及获得实施经验后推广总结，全面收获咨询成果

2. 管理咨询的流程

3. 诊与疗——矛盾的动态性、认识的层次性

1）先诊后疗

辨识并确定问题（去粗取精，去伪存真）→分析问题（问题的形成和发展机理——由表及里、由此及彼）→解决问题（三种情况三种打法：能直接治本的直接治本，暂时不能直接治本的用治标控制问题的范围、规模、程度，为寻求治本方案争取时间，在能力及成本允许的情况下标本兼治）

2）诊中有疗

及时控制问题的进程和危害，低成本的解决能够解决的问题，注意本阶段的主要矛盾是“诊”

3）疗中有诊

新的问题层出不穷（矛盾的连续性和多发性），认识问题的环境、手段、方法发生了变化，深层次问题的可认识性提高了

4. 从复杂的现象中正确辨识真正的问题

是技术问题吗？独立系统问题还是关联系统问题？

【思考题】
怎样理解“先诊后疗”“诊中有疗”“疗中有诊”

2-1 案例：药贩子，还是郎中——一切回归专业：专业医生和专业药师

1. CIM-OSA（开放体系结构）指出企业信息化解决方案的三种情况：通用、部分通用、专用

1）通用：OA（办公自动化）：文件流程、会议室借用、车辆调度、接待审批

2）部分通用：石化行业的生产优化系统

3）专用：个性化、定制化解决方案

2. 忌：情况不明决心大、问题不清办法多

3. 企业诊断：发现“不正常”、“不合理”

1）发现问题：

  • 对比分析找到“不正常”（存在可供比对的理想模型）

  • 演绎分析找到“不合理”（存在已知的推演逻辑）

2）发现关键问题：在多个“不正常”“不合理”中，判断居于关键位置的“不正常”“不合理”

3）发现科学问题：从特殊性中找到一般性

4. 管理咨询服务的收费

1）竞争性咨询服务的价格

  • 经过充分竞争的管理咨询服务

  • 某些专项咨询服务内容归一化、质量归一化，如人力资源管理培训、现场5S管理培训等

2）非竞争性咨询服务的价格

  • 价格与品牌、产品密切相关

  • 新趋势：方案免费、实施收费
